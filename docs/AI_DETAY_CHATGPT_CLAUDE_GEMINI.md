# 1. ChatGPT (OpenAI)

**Güncel Model:** GPT-5.2 (Aralık 2025)
**Slogan:** "Yapay zekanın herkes için erişilebilir hali"

Dosyalarını ve talimatlarını yükle, Claude'u özel bilgi tabanınla çalıştır. Proje bazlı hafıza ve bağlam yönetimi.

### MCP (Model Context Protocol)
Claude'un en güçlü silahlarından biri:
- **Nedir?** AI için evrensel adaptör — dış araçlar, uygulamalar ve veri kaynaklarıyla standart bağlantı
- **Mimari:** Host (Claude Desktop/Code) → Client → Server modeli
- **Popüler Entegrasyonlar:** GitHub, PostgreSQL, Git, Google Drive, Slack, Brave Search, dosya sistemi
- **MCP Tool Search:** Otomatik lazy loading ile bağlam kullanımında %95 azalma
- **Taşıma Seçenekleri:** Remote HTTP (önerilen), SSE, Yerel stdio

### 200K+ Context Window Avantajı

- ~150.000 kelimelik bir kitabı tek seferde analiz edebilir
- Tüm bir kod tabanını bağlam olarak verebilirsin
- 500+ sayfalık hukuki dokümanı özetleyebilir
- Birden fazla kaynağı karşılaştırmalı analiz edebilir
