## OTOMASYON

### 1. Zapier
- **Ne yapar:** 7.000+ uygulama arasında otomasyon. Tetikleyici → aksiyon mantığıyla kod yazmadan iş akışları kurar.
- **Nasıl para kazandırır:** Otomasyon kurulum hizmeti (proje 1.000-10.000₺), aylık bakım retainer, KOBİ dijitalleşme danışmanlığı, e-ticaret otomasyon paketleri, CRM entegrasyonu
- **Danışmanlık:** "İş süreçlerini otomatize et" paketi — müşterinin tekrarlayan görevlerini analiz edip Zapier ile otomatize etme
- **Avantajları:** En büyük uygulama ekosistemi (7.000+), kod gerektirmez, güvenilir, AI Actions
- **Dezavantajları:** Yüksek hacimde pahalı ($20/ay'dan $750+/ay'a çıkabilir), karmaşık logic sınırlı, vendor lock-in
- **Yazılım dili:** Python tabanlı backend, REST API
- **Fiyat:** Free (100 task/ay) → Starter $20/ay → Professional $49/ay → Team $69/ay → Enterprise özel
- **Türkçe:** Arayüz İngilizce
- **DZ Kitap:** Kitap 5 (Gelir), Kitap 7 (Otomasyon Mimarı — ana kaynak)

### 2. Make (eski Integromat)
- **Ne yapar:** Görsel otomasyon platformu. Zapier'e göre daha karmaşık senaryolar, daha uygun fiyat.
- **Nasıl para kazandırır:** Aynı hizmetler + daha kompleks entegrasyonlar, veri dönüşüm hizmetleri
- **Avantajları:** Görsel scenario builder, Zapier'den ucuz, daha esnek data mapping, HTTP/webhook desteği
- **Dezavantajları:** Öğrenme eğrisi Zapier'den yüksek, uygulama sayısı daha az (~2.000)
- **Yazılım dili:** Node.js tabanlı
- **Fiyat:** Free (1.000 ops/ay) → Core $9/ay → Pro $16/ay → Teams $29/ay → Enterprise özel
- **Türkçe:** Yok
- **DZ Kitap:** Kitap 7

### 3. N8N
- **Ne yapar:** Açık kaynak otomasyon platformu. Self-hosted veya cloud. Zapier/Make alternatifi, sınırsız kullanım.
- **Nasıl para kazandırır:** Kendi sunucunda sınırsız otomasyon, veri gizliliği gerektiren müşteriler için ideal, N8N kurulum+yönetim hizmeti
- **Avantajları:** Açık kaynak (ücretsiz self-hosted), sınırsız workflow, kod yazma imkanı (JavaScript/Python), güçlü AI node'ları
- **Dezavantajları:** Self-hosted sunucu yönetimi gerekir, topluluk Zapier/Make kadar büyük değil
- **Yazılım dili:** TypeScript/Node.js, Docker ile deployment
- **Fiyat:** Self-hosted FREE → Cloud Starter $20/ay → Pro $50/ay
- **Türkçe:** Yok
- **DZ Kitap:** Kitap 7

### 4. Apify
- **Ne yapar:** Web scraping ve veri çekme platformu. 1.500+ hazır scraper (Actor), API erişimi.
- **Nasıl para kazandırır:** Veri toplama hizmeti, fiyat izleme, rakip monitoring, lead listesi oluşturma
- **Avantajları:** Hazır scraper kütüphanesi, headless browser, proxy yönetimi, API
- **Dezavantajları:** Teknik bilgi gerektirebilir, bazı siteler scraping'i engeller
- **Fiyat:** Free ($5 kredi/ay) → Starter $49/ay → Scale $499/ay
- **DZ Kitap:** Kitap 7

### 5. Browse AI
- **Ne yapar:** No-code web scraping. Tarayıcı üzerinden tıkla-seç mantığıyla veri çekme.
- **Avantajları:** Kod gerektirmez, görsel seçim, otomatik zamanlama
- **Dezavantajları:** Karmaşık siteler için sınırlı, Apify kadar esnek değil
- **Fiyat:** Free (sınırlı) → Starter $39/ay → Professional $99/ay
- **DZ Kitap:** Kitap 7

## YAZILIM GELİŞTİRME / AI KODLAMA

### 6. Cursor AI
- **Ne yapar:** VS Code tabanlı AI-native kod editörü. Kod tamamlama, chat, multi-file edit, MCP desteği.
- **Nasıl para kazandırır:** Yazılım geliştirme hızını 3-5x artırma, freelance geliştirme, SaaS geliştirme, müşteri projeleri
- **Danışmanlık:** "AI destekli geliştirme" eğitimi, şirketlere Cursor entegrasyonu
- **Avantajları:** VS Code uyumlu, çok modelli (Claude/GPT/Gemini), Composer çok dosyalı edit, Agent mode, MCP
- **Dezavantajları:** Abonelik gerektiriyor ($20/ay), VS Code eklentileri kısmen uyumsuz olabilir
- **Yazılım dili:** Electron/TypeScript (editör), tüm programlama dillerini destekler
- **Fiyat:** Free (sınırlı) → Pro $20/ay → Business $40/ay/kişi
- **Türkçe:** Yok
- **DZ Kitap:** Kitap 3 (araç olarak), Kitap 7 (geliştirme stratejisi)

### 7. GitHub Copilot
- **Ne yapar:** GitHub'ın AI kod asistanı. VS Code, JetBrains, Neovim entegrasyonu. Kod tamamlama, chat, PR özetleme.
- **Avantajları:** GitHub entegrasyonu, geniş dil desteği, kurumsal güvenlik, Copilot Workspace
- **Dezavantajları:** Cursor kadar agresif değil, editör bağımsız değil
- **Fiyat:** Individual $10/ay → Business $19/ay/kişi → Enterprise $39/ay/kişi
- **DZ Kitap:** Kitap 3

### 8. Bolt.new
- **Ne yapar:** Tarayıcıda tam proje oluşturma. Prompt ile React/Next.js/Vue uygulaması üretir, preview + deploy.
- **Nasıl para kazandırır:** Hızlı MVP geliştirme hizmeti, prototip oluşturma, landing page
- **Avantajları:** Sıfırdan tam uygulama, tarayıcıda çalışır, Netlify deploy, çok hızlı
- **Dezavantajları:** Karmaşık projeler için sınırlı, production-ready olmayabilir
- **Yazılım dili:** React, Next.js, Vue, Svelte, TypeScript
- **Fiyat:** Free (sınırlı) → Pro $20/ay → Team özel
- **DZ Kitap:** Kitap 3, Kitap 5 (hızlı MVP)

### 9. Lovable.dev
- **Ne yapar:** AI ile hızlı prototip ve web uygulaması oluşturma. Supabase backend entegrasyonu.
- **Avantajları:** Supabase (veritabanı + auth) otomatik entegrasyon, hızlı prototipleme, güzel UI
- **Dezavantajları:** Bolt'a göre daha dar odak, erken aşama
- **Fiyat:** Free → Pro $20/ay
- **DZ Kitap:** Kitap 3, Kitap 5

### 10. Replit
- **Ne yapar:** Bulut IDE + AI Agent. Tarayıcıda kod yazma, çalıştırma, deploy. Replit Agent ile doğal dilde proje oluşturma.
- **Avantajları:** Tarayıcıda tam geliştirme ortamı, 50+ dil, Ghostwriter AI, anında deploy
- **Dezavantajları:** Performans sınırlı (bulut tabanlı), büyük projeler için yetersiz
- **Fiyat:** Free → Hacker $7/ay → Pro $20/ay → Teams $25/ay/kişi
- **DZ Kitap:** Kitap 3

### 11. Windsurf (Codeium)
- **Ne yapar:** AI-native kod editörü. Cursor rakibi. Cascade (multi-step AI) özelliği.
- **Avantajları:** Cascade çok adımlı reasoning, uygun fiyat, hızlı gelişim
