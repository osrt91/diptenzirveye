## 1. 10 Kitap Seri Haritası

### Seri Omurgası

Her kitap bir öncekinin yarattığı kapasiteyi alır ve bir üst katmana taşır:

```
Anlarsın → Konuşursun → Seçersin → Odaklanırsın → Kazanırsın
    → Üretirsin → Otomatize edersin → Ölçeklersin
        → Liderlik edersin → Çağı yönetirsin
```

### Sıralama Felsefesi

```
Faz 1 — Temel İnşa (Kitap 1–3)
  AI'ı anla → AI ile konuşmayı öğren → Doğru araçları seç

Faz 2 — Güç Kazanımı (Kitap 4–6)
  Odağını kur → İlk gelirini kazan → İçerik üretim sistemi kur

Faz 3 — Ölçeklendirme (Kitap 7–9)
  Otomasyona geç → Ekiple ölçekle → Organizasyonu dönüştür

Faz 4 — Entegrasyon (Kitap 10)
  Sektörünü yeniden tanımla → AI çağının mimarı ol
```
