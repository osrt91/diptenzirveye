## E-TİCARET PLATFORMLARI

### 1. Shopify
- **Ne yapar:** Uluslararası e-ticaret platformu. Online mağaza, ödeme, envanter, pazarlama, Shopify Magic (AI).
- **Nasıl para kazandırır:** Shopify mağaza kurulum hizmeti (3.000-15.000₺), tema özelleştirme, uygulama geliştirme, dropshipping mağazası, dijital ürün satışı, danışmanlık
- **Avantajları:** En popüler e-ticaret platformu, uygulama ekosistemi devasa (8.000+ app), global ödeme, Shopify Magic AI
- **Dezavantajları:** Aylık maliyet + işlem komisyonu, Türkiye'de ödeme entegrasyonu ek çaba gerektirir, vendor lock-in
- **Fiyat:** Basic $39/ay → Shopify $105/ay → Advanced $399/ay + %0.5-2% işlem komisyonu
- **Türkçe:** Kısmen (tema/arayüz çevrilebilir)
- **DZ Kitap:** Kitap 5 (Gelir — mağaza kurma)

### 2. WooCommerce
- **Ne yapar:** WordPress üzerine kurulu açık kaynak e-ticaret eklentisi. Tam kontrol, sınırsız özelleştirme.
- **Nasıl para kazandırır:** WooCommerce site kurulum hizmeti (5.000-30.000₺), plugin geliştirme, bakım/destek retainer, DİPTENZİRVEYE™ panelinin e-ticaret altyapısı
- **Avantajları:** Açık kaynak (ücretsiz), WordPress ekosistemi, tam kontrol, Türk ödeme entegrasyonları
- **Dezavantajları:** Teknik bilgi gerektirir, hosting ayrı, güvenlik/güncelleme sorumluluğu
- **Fiyat:** Plugin ücretsiz → Hosting $10-50/ay + Eklentiler değişken
- **Türkçe:** Tam destek
- **DZ Kitap:** Kitap 5, PART4 (DZ panel altyapısı)

### 3. Shopier
- **Ne yapar:** Türkiye odaklı dijital ürün satış platformu. Hızlı kurulum, entegre ödeme, otomatik teslimat.
- **Nasıl para kazandırır:** Dijital ürün satışı (e-kitap, şablon, prompt paketi), fiziksel ürün, hızlı test satışı
- **Avantajları:** Türk ödeme altyapısı hazır, hızlı kurulum (5 dk), otomatik dijital teslimat, sosyal medya linki
- **Dezavantajları:** Sınırlı özelleştirme, profesyonel mağaza hissi vermez, %3-5 komisyon
- **Fiyat:** Ücretsiz (komisyon bazlı) → Pro paketler mevcut
- **Türkçe:** Tam
- **DZ Kitap:** Kitap 5

### 4. Gumroad
- **Ne yapar:** Dijital ürün satış platformu. E-kitap, kurs, yazılım, müzik, şablon satışı.
- **Nasıl para kazandırır:** Dijital ürün satışı (global pazar), abonelik ürünleri, üyelik
- **Avantajları:** Basit kurulum, global ödeme (Stripe/PayPal), dijital teslimat otomatik, üyelik desteği
- **Dezavantajları:** %10 komisyon, Türkiye'den para çekme zorluğu (Wise gerekebilir)
- **Fiyat:** Ücretsiz (satış başına %10 komisyon)
- **Türkçe:** Yok
- **DZ Kitap:** Kitap 5

### 5. Ko-fi
- **Ne yapar:** İçerik üreticileri için destek + dijital ürün satış platformu. "Buy me a coffee" konsepti.
- **Avantajları:** 0% komisyon (Gold hariç), kolay kurulum, destek + satış bir arada
- **Fiyat:** Free (0% komisyon) → Gold $6/ay (ek özellikler)
- **DZ Kitap:** Kitap 5
